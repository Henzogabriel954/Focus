# Projeto Focus/Pomodoro

Bem-vindo ao projeto Focus/Pomodoro!

Este é um aplicativo simples para ajudar você a gerenciar seu tempo usando 
a técnica Pomodoro.

## Instalação

Para instalar e usar o aplicativo no seu Mac, siga estas instruções:

1.  **Clique no link abaixo para baixar o arquivo de instalação 
(`.dmg`):**

    [BAIXAR 
Focus.dmg](https://raw.githubusercontent.com/Henzogabriel954/Focus/refs/heads/main/APP/Focus.dmg)

2.  Após o download, abra o arquivo `Focus.dmg`.
3.  Arraste o ícone do aplicativo para a pasta `Aplicativos` 
(Applications) do seu Mac.
4.  Execute o aplicativo a partir da sua pasta de Aplicativos.
